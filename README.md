# Mesh Messenger

A decentralized peer-to-peer mesh messaging application designed for resilient communication without relying on centralized infrastructure.

## Overview

Mesh Messenger enables devices to communicate directly with one another using mesh networking principles. Messages can hop between nearby peers, creating a distributed communication network that can function even when traditional internet or cellular services are unavailable.

This project explores:

- Peer-to-peer communication
- Mesh networking
- Offline-first messaging
- Distributed systems
- Privacy-focused communication
- Resilient infrastructure

---

## Features

- Direct device-to-device messaging
- Multi-hop message relay
- Offline communication support
- Lightweight decentralized architecture
- No centralized server dependency
- Experimental mesh routing logic
- Open-source and community-driven

## Installation

Clone the repository:

bash git clone https://github.com/Mattjhagen/mesh-messenger.git cd mesh-messenger 

Install dependencies:

bash npm install 

Start the development server:

bash npm run dev

## Project Goals

The vision behind Mesh Messenger is to create communication tools that remain functional during:

- Internet outages
- Natural disasters
- Infrastructure failures
- Remote/off-grid environments
- High-censorship environments

The long-term goal is resilient communication powered by people, not centralized systems.
